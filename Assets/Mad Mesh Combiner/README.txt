== Mad Mesh Combiner ==

http://madpixelmachine.com/
Support: support@madpixelmachine.com

Feel free to write to me anytime if you have any troubles with using Mad Mesh Combiner!

There's a 1.1.5 version included, so if you will encounter issues with the current version, please try the previous one.

=== Features ===

- Atlases support
- Combines multiple materials into one
- Tiled meshes support
- Easy switching between source and combined objects
- Ability to adjust object position even after combine
- Option to leave source scripts enabled
- Normal maps support
- Submeshes support.
- No scripting abilities required!
- Super easy: select and combine!
